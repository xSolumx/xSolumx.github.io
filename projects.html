Projects

Additional-Blocks

Minecraft Plugin That Allows The Use Of Custom Modelled Blocks, Written In Java

I created a system that allowed me to add custom blocks into the base game of Minecraft, I wrote scripts to import JSON files and PNG files and appropriately catagorized them into their block or item types. Directional placement was a small challenge that I had to overcome using the players camera position and abstracted vector maths.

Rebels-Items

Framework To Implement Customized In Game Weapons and Tools, Written In Java

This is an old project that i wish i could've completed given the value it would have added. The solution uses some janky methods to store data on the item object itself which was not possible using the minecraft server framework at the time, so a workaround was required. this allowed me to easily view them within a customized interface and to manipulate the objects further as well.

Rebels-Back

Minecraft Plugin That Manipulates And Stores Player Locations

Coded in Java, this plugin stores the players location upon their death or any teleport event so that they can jump between locations for convenience, while this is a very simple plugin, it created features that did not exist at the time within other publically available sources.

Rebels-Pack

Minecraft Resource Pack - UI

Contains custom item icons, models, and modified JSON files. This integrates into Rebels-Items and Additional-Blocks to provide the customized textures and models. There was very little programming involved with this, though I did create 12 custom GUI's, 100 Custom icons, and 20 custom models for this project and integrate them with scripts.
